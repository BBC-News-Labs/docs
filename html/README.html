This documentation covers BBC News Labs services, including the Juicer API.

Juicer Documentation

The Juicer Documentation covers to get started usign the BBC News Labs Juicer API. You will need an API key of your own to make calls to the API.

Juicer Client Libraries

  • There is a Juicer client library for node.js called newsQuery
  • There is also a Juicer client GEM for Ruby
